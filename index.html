איך זה עובד? ממלאים כמה כסף יש היום בעו"ש, מה ה"אפס החדש" (הסכום שתמיד נשאר בעו"ש כרשת ביטחון), ואת ההוצאות שנשארו עד המשכורת הבאה. המערכת מחשבת כמה כסף פנוי יש החודש — את הסכום הזה מעבירים לחיסכון או להשקעה לפי המטרות שלמטה.

✅ כל משתמש — גישה לנתונים שלו בלבד, מאובטח ע"י Firebase מצד השרת
✅ תום אני (info@tomani.co) — גישת קריאה לכל המשתמשים
✅ איפוס סיסמה — לחץ "שלח איפוס סיסמה" ליד כל לקוח
כדי שהאבטחה תעבוד — ודא שה-Firestore Rules מוגדרים נכון:
match /users/{userId} {
  allow read, write: if request.auth != null &&
    (request.auth.uid == userId ||
     request.auth.token.email == 'info@tomani.co');
}
match /users/{userId}/{document=**} {
  allow read, write: if request.auth != null &&
    (request.auth.uid == userId ||
     request.auth.token.email == 'info@tomani.co');
}
